# PseudoInstaller

A CLI tool that simulates installing packages with no real system impact. It generates realistic progress output for entertainment purposes.

This project is inspired by https://github.com/buyukakyuz/install-nothing

## Setup

```
git clone https://github.com/PearlynnT/pseudo-installer.git
cd pseudo-installer
```

## Running the scripts

To run the `.py` scripts

```
python <filename>
```

To run the `.js` scripts

```
node <filename>
```
